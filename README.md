# Impetus TechBlog
**I have build a Techblog using JSP, Servlet and MySQL database.**
**Go to Branch** **|->Impetus-blog**
      

## Tools and technologies used
* JSP - 2.2 +
* IDE - Eclipse 
* JDK - 1.8 or later
* Apache Tomcat - 9
* JSTL - 1.2.1
* Servlet API - 4
* MySQL - mysql-connector-java-8.0.13.jar
